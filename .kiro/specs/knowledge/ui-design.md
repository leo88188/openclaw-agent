## 2. 页面清单与导航

### 2.1 全局导航（左侧边栏）

- 宽度: 240px（desktop）/ 60px 图标模式（tablet）/ 隐藏+汉堡菜单（mobile）
- 背景: var(--surface)
- 当前页高亮: 左侧 3px var(--primary) 边框 + 背景 var(--primary-light)
- 折叠/展开: 点击底部 toggle 按钮，状态存 localStorage

### 2.2 页面路由

| 页面 | 路由 | 优先级 | 导航图标 |
|------|------|--------|---------|
| 仪表盘 | `/` | P2 | fa-dashboard |
| 知识管理 | `/knowledge` | P0 | fa-book |
| 语义搜索 | `/search` | P0 | fa-search |

| 元数据浏览 | `/metadata` | P1 | fa-database |

### 2.3 页面间导航关系

- 仪表盘「最近知识」→ 点击跳转 `/knowledge`（带 id 参数打开详情 Modal）
- 仪表盘「快捷操作」→ 新建知识 → `/knowledge`（自动打开创建 Modal）
- 搜索结果 → 点击结果卡片 → 根据类型跳转对应页面
- 所有页面 → 顶部搜索框 → `/search`（带 query 参数）

## 3. 核心页面交互设计

### 3.1 仪表盘 `/` (P2)

**布局：** 上下结构
- 顶部: 4 个统计卡片（知识总数、搜索次数、活跃用户、Skill 数量）横向排列
- 中部: 快捷操作栏（新建知识、搜索、导入元数据）3 个按钮
- 底部: 最近知识列表（最近 10 条，表格形式）

**交互：**
- 统计卡片: 数字从 0 动画递增到实际值（300ms ease-out）
- 快捷操作: hover 时卡片微上浮 translateY(-2px)
- 最近列表: 点击行 → 跳转 `/knowledge` 并打开该条目详情

**状态：**
- Loading: 4 个卡片骨架屏 + 列表骨架屏（5 行）
- Error: 居中错误提示 + 重试按钮
- Empty: 「还没有知识条目，立即创建第一条」+ CTA 按钮

### 3.2 知识管理 `/knowledge` (P0)

**布局：** 顶部操作栏 + 内容列表

```
┌─────────────────────────────────────────────┐
│ [分类下拉▼] [标签筛选▼]    [搜索框🔍]  [+ 新建] │  ← 操作栏
├─────────────────────────────────────────────┤
│ ┌─────────┐ ┌─────────┐ ┌─────────┐        │
│ │ 知识卡片 │ │ 知识卡片 │ │ 知识卡片 │        │  ← 卡片网格
│ │ 标题     │ │ 标题     │ │ 标题     │        │     desktop: 3列
│ │ 摘要...  │ │ 摘要...  │ │ 摘要...  │        │     tablet: 2列
│ │ 分类 标签│ │ 分类 标签│ │ 分类 标签│        │     mobile: 1列
│ │ 时间     │ │ 时间     │ │ 时间     │        │
│ └─────────┘ └─────────┘ └─────────┘        │
│                                             │
│              [1] [2] [3] ... [>]             │  ← 分页
└─────────────────────────────────────────────┘
```

**操作栏交互：**
- 分类下拉: 单选，选中后立即触发列表刷新，显示选中分类名
- 标签筛选: 多选下拉，选中标签显示为 chip，可逐个移除
- 搜索框: 输入后 300ms 防抖触发关键词搜索（非向量搜索）
- 新建按钮: 打开创建 Modal

**知识卡片交互：**
- 展示: 标题（H3 截断 1 行）+ 摘要（正文截断 2 行）+ 分类标签 + 更新时间
- Hover: box-shadow 加深，cursor: pointer
- 点击: 打开详情 Modal
- 卡片右上角: 更多操作按钮（⋯），hover 显示下拉菜单（编辑、删除）

**创建/编辑 Modal：**
- 宽度: 720px
- 字段:
  - 标题输入框（必填，placeholder: "输入知识标题"）
  - Markdown 编辑器（必填，左右分栏：编辑 | 预览）
  - 分类选择（下拉，可输入新分类）
  - 标签输入（输入后按 Enter 添加 chip，可删除）
- 底部: [取消] [保存] 按钮
- 保存流程: 按钮 loading → API 请求 → 成功 Toast + 关闭 Modal + 刷新列表 / 失败 Toast
- 未保存退出: 如有修改，弹出确认弹窗「有未保存的修改，确定离开？」

**详情 Modal：**
- 宽度: 720px
- 顶部: 标题 + 分类标签 + 创建时间 + 更新时间
- 内容: Markdown 渲染（只读）
- 底部: [编辑] [删除] 按钮
- 编辑: 切换为编辑 Modal
- 删除: 弹出确认弹窗 → 确认后 API 删除 → 成功 Toast + 关闭 + 刷新列表

**删除确认弹窗：**
- 标题: 「确认删除」
- 内容: 「确定要删除「{标题}」吗？此操作不可撤销。」
- 按钮: [取消] [删除]（红色）
- 删除按钮 loading 态防重复点击

**分页：**
- 每页 12 条（4行×3列）
- 页码按钮 + 上一页/下一页
- 切换页码: 列表区域显示 loading bar（顶部细条），不清空已有内容

### 3.3 语义搜索 `/search` (P0)

**布局：** 居中搜索 + 左侧过滤 + 右侧结果

```
初始状态（无搜索词）:
┌─────────────────────────────────────────────┐
│                                             │
│           🔍 搜索知识库                       │
│     ┌──────────────────────────────┐        │
│     │ 输入自然语言描述...           │        │  ← 大搜索框居中
│     └──────────────────────────────┘        │
│                                             │
│     热门搜索: [订单状态] [用户表] [SQL优化]   │  ← 热门标签
│                                             │
└─────────────────────────────────────────────┘

有结果状态:
┌──────────┬──────────────────────────────────┐
│ 过滤面板  │  搜索框 [输入内容...]              │
│          │                                  │
│ 分类      │  找到 23 条结果（0.8s）            │
│ ☐ 业务知识│                                  │
│ ☐ 技术文档│  ┌──────────────────────────────┐│
│ ☐ 元数据  │  │ 结果标题                      ││
│          │  │ ...匹配内容高亮摘要...          ││
│ 标签      │  │ 分类 · 标签 · 相关度 92%      ││
│ ☐ MySQL  │  └──────────────────────────────┘│
│ ☐ Redis  │                                  │
│          │  ┌──────────────────────────────┐│
│ 时间范围  │  │ 结果标题                      ││
│ [最近7天▼]│  │ ...匹配内容高亮摘要...          ││
│          │  └──────────────────────────────┘│
└──────────┴──────────────────────────────────┘
```

**搜索框交互：**
- 初始: 页面居中大搜索框，高度 56px，圆角 28px
- 输入后: 搜索框上移到顶部，过滤面板从左侧滑入
- 防抖: 停止输入 300ms 后自动搜索
- 快捷键: 按 Enter 立即搜索
- 清空: 输入框右侧 × 按钮，点击清空并回到初始状态

**过滤面板交互：**
- 分类: checkbox 多选，选中后立即触发重新搜索
- 标签: checkbox 多选
- 时间范围: 下拉选择（全部 / 最近 7 天 / 最近 30 天 / 最近 90 天）
- 重置: 底部「清除所有筛选」链接

**搜索结果交互：**
- 结果卡片: 标题 + 高亮匹配摘要（关键词用 `<mark>` 标签高亮）+ 分类 + 相关度百分比
- 点击结果: 打开详情 Modal（复用知识详情 Modal）
- 加载更多: 滚动到底部自动加载下一页（infinite scroll）

**状态：**
- Searching: 搜索框下方 loading bar + 结果区域骨架屏
- No Results: 居中插画 + 「没有找到相关结果」+ 建议（尝试不同关键词、减少筛选条件）
- Error: Toast 错误提示 + 重试按钮

```
┌─────────────────────────────────────────────┐
│ [全部] [数据分析] [代码生成] [文案写作] [+ 创建] │  ← 分类 Tab
├─────────────────────────────────────────────┤

│ │ 分类      │ │ 分类      │ │ 分类      │     │
│ │ ❤️ 12 👁 56│ │ ❤️ 8  👁 34│ │ ❤️ 23 👁 89│     │
│ └──────────┘ └──────────┘ └──────────┘     │
└─────────────────────────────────────────────┘
```

**分类 Tab 交互：**
- 横向滚动（移动端），选中 Tab 下方 2px var(--primary) 指示线
- 切换 Tab: 列表区域 loading bar + 重新加载

**Skill 卡片交互：**
- 展示: 图标 + 名称 + 描述（截断 2 行）+ 分类标签 + 收藏数 + 使用数
- Hover: 微上浮 + 阴影加深
- 点击: 打开 Skill 详情 Modal

**Skill 详情 Modal：**
- 宽度: 640px
- 内容: 名称、描述、提示词模板（代码块展示）、参数说明、使用统计
- 操作:
  - [❤️ 收藏/取消收藏] — 点击切换，即时更新图标和计数（乐观更新）
  - [使用] — 展开参数填写区域
  - [编辑]（仅作者可见）— 打开编辑 Modal
- 使用流程: 点击「使用」→ 显示参数输入表单 → 填写参数 → 点击「生成」→ 生成最终提示词 → 一键复制

**创建 Skill Modal：**
- 字段: 名称（必填）、描述、分类（下拉）、提示词模板（必填，支持 `{{参数名}}` 占位符）、是否公开（toggle）
- 保存流程同知识创建

## 5. 动效规范

### 5.1 页面切换

- 方式: 多页应用，无 SPA 过渡
- 页面加载: 内容区域 fadeIn 200ms ease-out

### 5.2 元素动效

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| Modal 打开 | fadeIn + translateY(20px→0) | 200ms | ease-out |
| Modal 关闭 | fadeOut + translateY(0→10px) | 150ms | ease-in |
| Toast 进入 | translateX(100%→0) | 200ms | ease-out |
| Toast 退出 | opacity(1→0) | 150ms | ease-in |
| 卡片 hover | translateY(0→-2px) + shadow 加深 | 150ms | ease-out |
| 按钮 active | scale(1→0.98) | 100ms | ease-out |
| 骨架屏 | 背景色 shimmer 动画 | 1.5s | linear infinite |
| 列表刷新 | 顶部 loading bar 从左到右 | 持续到完成 | linear |
| 数字递增 | 0 → 目标值 | 300ms | ease-out |

### 5.3 减弱动效

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 8. 全局交互规范

### 8.1 键盘导航

| 快捷键 | 作用域 | 行为 |
|--------|--------|------|
| Tab | 全局 | 焦点在可交互元素间移动 |
| Enter | 按钮/链接 | 触发点击 |
| ESC | Modal/下拉 | 关闭当前浮层 |
| / | 全局（非输入框） | 聚焦搜索框 |

### 8.2 Toast 规范

| 类型 | 图标 | 背景 | 持续时间 |
|------|------|------|---------|
| Success | fa-check-circle | #ecfdf5 | 3s |
| Error | fa-times-circle | #fef2f2 | 5s |
| Warning | fa-exclamation-triangle | #fffbeb | 4s |
| Info | fa-info-circle | #e6f2ff | 3s |

### 8.3 确认弹窗规范

仅以下操作需要确认弹窗：
- 删除知识条目
- 删除 Skill
- 移除团队成员
- 导入元数据（覆盖操作）

其他操作（创建、编辑、收藏）不需要确认。

### 8.4 空状态文案

| 页面 | 无数据文案 | CTA |
|------|----------|-----|
