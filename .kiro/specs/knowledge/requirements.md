## 1. 产品概述

Knowledge 是一个 AI 知识库管理平台，为 OpenClaw 龙虾各团队提供永久记忆管理能力。核心功能包括：对数据库元数据、表结构、业务指标、术语、团队业务知识的向量化存储与搜索，以及共享 Skill 和提示词的管理与分享。

## 2. 用户故事

### US-1: 知识条目管理
- 作为团队成员，我可以创建/编辑/删除知识条目，以便沉淀团队业务知识
- 验收标准：
  - 支持创建知识条目（标题、内容、分类、标签）
  - 支持编辑和软删除
  - 内容自动向量化存储
  - 支持 Markdown 格式

### US-2: 向量化搜索
- 作为用户，我可以通过自然语言搜索知识库，获得语义相关的结果
- 验收标准：
  - 输入自然语言查询，返回语义相关结果
  - 结果按相关度排序
  - 支持分类/标签过滤
  - 搜索响应时间 < 2s

### US-3: 数据库元数据管理
- 作为 DBA/开发者，我可以导入和浏览数据库元数据（表结构、字段说明、业务指标）
- 验收标准：
  - 支持导入 MySQL 表结构元数据
  - 自动提取字段名、类型、注释
  - 元数据自动向量化，支持语义搜索

### US-4: Skill/提示词管理
- 作为用户，我可以创建、分享和使用 Skill（提示词模板）
- 验收标准：
  - 支持创建 Skill（名称、描述、提示词模板、参数）
  - 支持公开/私有设置
  - 支持收藏和使用统计
  - 支持按分类浏览

### US-5: 团队空间
- 作为团队管理者，我可以管理团队的知识空间和成员权限
- 验收标准：
  - 支持创建团队空间
  - 支持成员邀请和角色管理（管理员/编辑者/查看者）
  - 团队内知识隔离

## 3. 功能模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 知识管理 | CRUD、分类、标签、Markdown编辑 | P0 |
| 向量搜索 | 语义搜索、过滤、排序 | P0 |
| 元数据管理 | 数据库表结构导入、浏览、搜索 | P1 |
| Skill管理 | 提示词CRUD、分享、收藏 | P1 |
| 团队空间 | 空间管理、成员权限 | P2 |
| 仪表盘 | 统计概览、最近活动 | P2 |

## 4. 技术约束

### 前端
- 纯 HTML + TailwindCSS CDN + jQuery
- 企业级后台风格，简洁清爽高端
- Grid 8列布局，不使用 SVG 连线
- 流程节点横向连接线（绿色完成/蓝色进行中/灰色未开始）
- 保持动画：跳动、呼吸、摇摆
- CDN 依赖：tailwindcss、font-awesome 4.7、jQuery 3.7.1

### 后端
- Python FastAPI
- MySQL 数据库（库名：knowledge）
- Redis 缓存和实时状态管理
- RESTful API，支持跨域 (CORS)

### 架构
- 前后端分离，不同目录
- 后端脚本加载静态 HTML
- jQuery AJAX 调用 API
- JS 拆分：api.js（接口层）、组件 JS
